Discovery in the REPL

The goal of this page is to learn how to discover Python commands in the REPL and trying them out. The real strength of Pythong is that there is a library you can import for just about anything online. We will start with one of the most useful.
# Goal Try This What to Observe
1 Import a library import time Loads a number of methods for timing
2 Basic autocomplete Use the TAB button to finish what you type time. and then press tab See all the possible methods?
3 Getting help try help(time.time) What result do you get?
