# Goal Try This What to Observe
1 Learn Python interactively in a REPL Get started with basic commands Execute code and see immediate results
2 Basic arithmetic 2+2 then press return What result do you get?
3 Variable assignment a = 2 then just type aa What result do you get?
4 Variable usage now try typing a + 2 What was returned?
5 Printing you can print by typing print('a is ',a) you can also try print(f'a is {a}') - and place the variable value in the text (note the f for format)
6 Running multiple times try running a loop:

import time
for i in range(10):
    print(i)
    time.sleep(1)
 the for loop should run 10 times
Note that when you run code in the REPL, the webpage will not respond until it is done
