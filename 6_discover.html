Self-Help

Python has built-in tools that allow users to explore the different commands in a library. Type the text below into the REPL and press the TAB button. What do you see?

  • motor.

The REPL should print out a menu of different commands available for the motor.

You can also use help() to learn more about certain hardware or commands. Try typing the below into the REPL.

  • help(motor)

The REPL should provide information about the motor and its sensor readings.
