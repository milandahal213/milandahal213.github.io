Using print()

The REPL can also write out requested information with print(). Try typing the lines below into the REPL.

  • a = 7
    print(a)

The REPL should print the current value of the variable "a."

You can also print custom text. This is useful when you want to track whether a piece of your program was able to run correctly. Try entering the line below into the REPL.

  • print('hello!')

Try printing out a different word or phrase.
