Interactive Python Learning Guide

# Goal Try This What to Observe
1 Learn Python interactively in a REPL Get started with basic commands Execute code and see immediate results
2 Basic arithmetic 2+2 then press return What result do you get?
3 Variable assignment a = 2 then a+2 What result do you get?
4 Connect to a channel channel.msg What message value do you receive?
5 Explore the channel API help(channel) Review available methods and how to use them
6 Connecting a motor Turn on and advertise Connect and rename as motor1
7 Reading position motor1.position You should see the position of the motor
8 Running the motor motor1.set_speed(100) motor1.run() You should see the motor spin
