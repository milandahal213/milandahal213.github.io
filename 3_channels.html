The goal of this page is to learn how channels work - they are a powerful way for any two webpages to talk to each other. We will use channels to help all the inventions work together.
# Goal Try This What to Observe
1 Post something on the channel (after you have connected) type channel.send('/LEGO','it worked') and hit return You should see that appear above in blue. So should anyone connected to the channel
2 Send only to a friend Change the topic (/LEGO) to something else - and tell your friend channel.send('/new','it is a secret') and then press tab Only those people that have your topic typed in will see your message.
3 Reading a channel the variable channel.msg will have the latest channel value What result do you get?
