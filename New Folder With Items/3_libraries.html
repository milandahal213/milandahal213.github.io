Python Libraries

The real strength of Python is there are a zillion libraries you can import and use to do anything you can imagine - and genAI is very good at writing Python code.

When you import a Python library, your program can use the library’s pre-written tools and functions.

Let’s try importing a library

  • import time

You imported the time library. It has many functions like current time, sleep, etc. Copy the entire code below into the REPL and press [ENTER].

  • import time
    print("Hi!")
    time.sleep(1)
    print("Hello after 1 second!")

Python has built-in tools that allow users to explore the different commands in a library. For example, pressing [TAB] after typing the library name and a dot (.) will show you all the available functions in that library.
